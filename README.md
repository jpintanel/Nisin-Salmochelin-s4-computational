# Nisin – Salmochelin S4 | Computational Analysis

Computational chemistry and cheminformatics analysis of a novel 
antimicrobial conjugate linking nisin (lantibiotic) to Salmochelin S4 
(siderophore) via click chemistry (triazole linker).

## Research Questions
- Does the conjugate exploit the IroN/TonB iron uptake system?
- Can nisin bind lipid II while intact (conjugated) or must it be released?

## Tools & Methods
Python · RDKit · SciPy · ColabFold · AutoDock Vina · OpenMM
